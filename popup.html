        Instruction for importing and exporting the bookmark's taxonomy

        If you want to export your bookmark's taxonomy you should click the 'Export' button and it will save a 'bookmarkstaxonomy.json' file in your 'Download' folder.

        If you want to import your bookmark's taxonomy you should replace the 'bookmarkstaxonomy.json' file in this plugin's folder with the 'bookmarkstaxonomy.json' file containing the bookmark's taxonomy you want to import and then click the 'Import' button.
