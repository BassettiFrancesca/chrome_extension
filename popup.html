        Import & Export bookmarks' taxonomy
        Instructions for importing and exporting the bookmarks' taxonomy

        If you want to export your bookmarks' taxonomy you should click the 'Export' button and it will save a 'bookmarkstaxonomy.json' file in your 'Download' folder.

        If you want to import a bookmarks' taxonomy you should replace the 'bookmarkstaxonomy.json' file in this plugin's folder with the 'bookmarkstaxonomy.json' file containing the bookmarks' taxonomy you want to import and then click the 'Import' button.

        Warning: if you import a bookmarks' taxonomy the bookmarks you have saved will be lost.
